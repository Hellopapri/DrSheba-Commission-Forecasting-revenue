# 🏥 DrSheba Commission Forecasting & Revenue Prediction System

A Django-based intelligent forecasting system for predicting doctor commission and hospital revenue using real healthcare data.

---

## 📌 Project Overview

This project analyzes healthcare booking data and predicts:

- 📊 Commission Revenue Forecast
- 📈 Multi-Level Forecasting
- 🔮 Single Booking Revenue Prediction
- 📉 Growth & Cancellation Impact Simulation

Built for data-driven decision-making in healthcare platforms.

---

## 🚀 Features

- Commission Revenue Forecasting
- Scenario Simulation (Growth % & Cancellation %)
- Multi-Level Forecast (Department / City / Doctor Type)
- Single Booking Revenue Prediction
- Interactive Dashboard View
- Excel Workbook Update Automation

---

## 🛠 Tech Stack

- Python
- Django
- Pandas
- NumPy
- SQLite
- Power BI (Analysis)
- Google Colab

---

## 📂 Dataset Files

- drsheba_doctors.csv  
- drsheba_hospitals.csv
